# Password Strength Checker

## Description
This is a Python-based Password Strength Checker that evaluates the strength of a password using different security factors.

## Features
- Checks password length
- Checks uppercase letters (A-Z)
- Checks lowercase letters (a-z)
- Checks numbers (0-9)
- Checks special characters
- Provides suggestions to improve password strength

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run

```bash
py password_checker.py
```

## Sample Output

```
Enter your password: Reera@2127

Password Strength: Strong Password
Your password is secure!
```
